apipost-tools 是一个一致性、模块化、高性能的 JavaScript 实用工具库。

# 安装

```shell
npm i apipost-tools
```

# 基础使用
需引入：

```js
import ATools from 'apipost-tools';
let json='{"name":"我是一名ApiPost用户"}';
let email='1030698842@qq.com';

ATools.isJson(json); //true
ATools.isEmail(email); //true
```
